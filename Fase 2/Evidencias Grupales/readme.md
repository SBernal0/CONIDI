# Evidencias Grupales — Fase 2

Descripción
-----------
Carpeta para evidencias que reflejen el trabajo en equipo en la Fase 2 (actas, entregables colectivos, resultados de pruebas grupales, retroalimentación).

Contenido recomendado
---------------------
- Documentos que evidencien coordinación y decisiones de equipo.
- Entregables conjuntos y reportes de pruebas o validaciones.
- Resumen de acuerdos y responsables cuando aplique.

Indicaciones
------------
Antes de subir, coordinar con el equipo la versión a presentar y agregar en el commit una breve descripción del contenido.
