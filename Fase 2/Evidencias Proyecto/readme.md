# Evidencias del Producto — Fase 2

Descripción
-----------
Carpeta para almacenar las evidencias relacionadas con el producto de software generadas en la Fase 2 del proyecto.

Contenido sugerido
------------------
- Entregables del desarrollo.
- Resultados de pruebas y verificación.
- Capturas o demostraciones de la interfaz.
- Paquetes o builds cuando correspondan.

Cómo usar esta carpeta
----------------------
Sube los archivos con un nombre que permita identificar su contenido y fecha. Añade en el mensaje de commit una breve descripción (qué es, autor, fecha).

Notas
-----
Evitar incluir datos sensibles sin anonimizar. Mantener la carpeta ordenada para facilitar la revisión.
